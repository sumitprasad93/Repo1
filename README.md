# Calculator Project

This project is a simple GUI calculator implemented in Python using the Tkinter library.

## Project Structure

- : This file contains the main application logic for the calculator, including the GUI and the calculation functions.
- : This file contains unit tests for the calculator's logic.
- : This file, providing information about the project and how to run it.

## How to Run the Calculator

To run the calculator, you need to have Python installed with Tkinter support (which is usually included with standard Python installations).

1.  Navigate to the directory containing the project files in your terminal.
2.  Execute the  script using the Python interpreter:

    ```bash
    python calculator_gui.py
    ```

This will launch the calculator window.
